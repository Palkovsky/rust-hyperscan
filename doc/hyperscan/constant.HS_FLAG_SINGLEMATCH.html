pub const HS_FLAG_SINGLEMATCH: u32 = 8

Compile flag: Set single-match only mode.

This flag sets the expression's match ID to match at most once. In streaming mode, this means that the expression will return only a single match over the lifetime of the stream, rather than reporting every match as per standard Hyperscan semantics. In block mode or vectored mode, only the first match for each invocation of @ref hs_scan() or @ref hs_scan_vector() will be returned.

If multiple expressions in the database share the same match ID, then they either must all specify @ref HS_FLAG_SINGLEMATCH or none of them specify @ref HS_FLAG_SINGLEMATCH. If a group of expressions sharing a match ID specify the flag, then at most one match with the match ID will be generated per stream.

Note: The use of this flag in combination with @ref HS_FLAG_SOM_LEFTMOST is not currently supported.
